Python,

Adding Posts, Tags, Comments, and Images, 
Post Search, 
Filter for the Latest Post, Most Commented Post, and Total Number of Posts, 
Filter for Tags, 
Similar Post, 
Lists of Posts from the most recent, 
Pagination, 
Post Thumbnail, and
Commenting System for each post
